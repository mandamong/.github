# mandamong (만다몽)

> **AI 를 이용해 만다라트 표를 쉽게 만들고, 목표에 빠르게 가까워지세요.**

<br>

만다라트(Mandal-Art)는 목표 달성을 위해 사용되는 유용한 도구입니다.  
하지만 막상 표를 채우려면 막막하게 느껴집니다.  

핵심 주제를 중심으로 연관성 있는 계획을 세우기 위해서는 많은 시간과 노력이 필요하기 때문인데요.  

**만다몽**은 이 부담스러운 과정을 AI 를 통해 해결합니다.
이제 더이상 계획에 시간을 쓰지 말고 **목표 달성**에 집중하세요.

## 💡 핵심 가치

만다몽은 AI 를 활용하여 만다라트 표 생성 과정을 단순화합니다.

**핵심 주제**만 입력하면 AI 가 **4개의 세부 목표**와 **20개의 실천 행동**을 제안합니다.

한 번의 클릭으로 만다라트 표를 완성하고, 바로 실행에 옮기세요.

## 🚀 시작하기

1. 새 만다라트를 만들고 **이루고 싶은 주제**를 입력하세요.  
2. `초안 만들기` 버튼을 눌러 AI 의 제안을 확인하세요.  
3. AI 의 제안을 자유롭게 수정하여 나만의 만다르트 표를 완성하세요.
4. 완료한 행동을 하나씩 체크해가며 주제와 가까워지세요.

## 📚 용어 정리

- **주제 (Subject)**: 만다라트의 중심에 적는 이루고 싶은 주제
- **목표 (Objective)**: 주제를 이루기 위한 4가지 세부 목표
- **행동 (Action)**: 각 목표를 실행으로 옮기기 위한 구체적인 5가지 실천 방법 (총 20개)
